Component with @components/* Import

The button component is imported using @components/ts-button.js

Default Custom Label Disabled

Note: Labels are automatically capitalized using the @utils/format utility!

Path Mappings Configuration

Defined in tsconfig.json:

"paths": {
    "@components/*": ["./src/components/*"],
    "@utils/*": ["./src/utils/*"]
    }

Usage Examples

// Instead of relative paths:
    import { TsButton } from '../../components/ts-button.js';
    import { capitalize } from '../../utils/format.js';

    // Use clean path aliases:
    import { TsButton } from '@components/ts-button.js';
    import { capitalize } from '@utils/format.js';
