# Slots Reference: `button-element`




## Available Slots




### Default slot

Button content






### `icon` slot

Button icon







## HTML Usage Examples



### Using default slot

```html
<button-element>
  Button content
</button-element>
```



### Using `icon` slot

```html
<button-element>
  <div slot="icon">Button icon</div>
</button-element>
```



### Complete example with all slots

```html
<button-element>
  Button content
  <div slot="icon">Button icon</div>
</button-element>
```
